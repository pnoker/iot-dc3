# 贡献者行为准则

## 我们的承诺

我们作为贡献者和维护者承诺，让 IoT DC3 社区的参与对所有人都是零骚扰的体验，不论年龄、体型、残疾、民族、性别认同与表达、经验水平、教育程度、国籍、个人外貌、种族、宗教、性认同与取向，或其他任何个人特质。

## 期望的行为

- 使用友好、包容的语言。
- 尊重不同的观点和经验。
- 虚心接受建设性反馈。
- 聚焦于对项目和用户最有利的事情。
- 对其他社区成员展现同理心。

## 不可接受的行为

不可接受的行为包括骚扰、恐吓、歧视性言论、带有性暗示的言论或图像、人身攻击、持续干扰、未经许可发布他人隐私信息，或维护者合理认为不适合专业社区的任何其他行为。

## 执行

项目维护者可以删除、编辑或拒绝不符合本行为准则的评论、提交、代码、议题、拉取请求、Wiki 编辑或其他贡献。维护者也可以临时或永久封禁对社区有害的参与者。

辱骂、骚扰或其他不可接受的行为可以通过创建 Issue 或联系一位或多位项目维护者举报。举报将被及时、公正地审查和调查。

本行为准则改编自 Contributor Covenant 2.1 版：
https://www.contributor-covenant.org/version/2/1/code_of_conduct/
