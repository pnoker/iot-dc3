IoT DC3 分层架构图

Application → Management → Data → Driver · 安全与负载 · 平台与系统 · AI闭环

四层架构

  • • Application — 客户端注册、开放API、数据分发、扩展模块
  • • Management — 调度、告警、通知、日志、消息
  • • Data — 存储、交互操作、设备库、数据管理与采集
  • • Driver — TCP / OPC / MODBUS / MQTT / PLC / 可扩展

安全与平台

  • • Security & Load — 负载均衡、安全套件、Token、加密
  • • Platform & System — 管理平台、容器化、快速部署
  • • Device & System & Monitoring — 设备/系统/监控一体化

数据流向

  • • Gateway / Cloud Server → 向下接入四层架构
  • • Data Layer → 交互操作与数据管理驱动 Driver 层
  • • Driver Layer ↔ 真实/虚拟设备、SDK 快速开发
  • • AI 闭环：数据采集 → 分析决策 → 指令回写 → 结果回流
