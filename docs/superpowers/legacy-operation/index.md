# 旧版操作手册归档

这里保留旧版 dc3-docs（VuePress 1.x）的截图式 Web UI 操作资料，便于迁移和对照历史界面。

::: tip
这些页面是历史归档。当前公开操作路径请阅读 [操作手册](../../operation/)；旧截图仅作为功能迁移和界面对照参考。
:::

## 目录

- [驱动](driver.md) — 查看驱动列表/详情，以及驱动关联的设备
- [模板](template.md) — 创建/编辑设备模板，配置基础信息与位号
- [设备](device.md) — 新增/导入设备，关联模板与位号，查看设备数据
- [数据](data.md) — 查看设备对应位号的实时数据

## 推荐阅读顺序

1. 先看 [模板](template.md)：理解如何按设备类型抽象通用配置。
2. 再看 [设备](device.md)：理解如何把模板实例化为具体设备并接入。
3. 最后看 [驱动](driver.md) 和 [数据](data.md)：查看接入后的状态与数据流。
