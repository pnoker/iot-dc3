## 1. Prepare

- `git`
- `Visual Studio Code`
- `nodejs` >= 18, recommended to use 22
- `pnpm`, install using `corepack enable && corepack prepare pnpm@latest --activate` (or `npm install -g pnpm`)

## 2. Source code

```bash
git clone https://github.com/pnoker/iot-dc3-web.git
```

## 3. Develop

```bash
cd iot-dc3-web

# install
pnpm install

# run
pnpm dev
```
